```bash
git clone https://github.com/t1tandr/uchat.git
cd uchat/
make
```

## Usage

```bash
git clone https://github.com/t1tandr/uchat.git
cd uchat/
make
```

Start the server
```bash
cd server/
./uchat-server <port-number>
```

Start the client
```bash
cd client/
./uchat <ip> <port>
```

To finish server
```bash
kill <server PID>
```
