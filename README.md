```bash
git clone https://github.com/t1tandr/uchat.git
cd uchat/
make
```

## Usage

```bash
git clone https://github.com/t1tandr/uchat.git
cd uchat/
make
```

Start server:
```bash
cd server/
./uchat-server 8080
```

Start client:
```bash
cd client/
./uchat localhost 8080
```

To shutdown server execute next command:
```bash
kill server-pid
```
